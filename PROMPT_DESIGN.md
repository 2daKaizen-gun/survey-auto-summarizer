# 🎯 AI Prompt Design - Batch Insight Model

## 1. 분석 목적
개별 요약을 넘어, 전체 응답자 데이터에서 [국적, 성별, 참석여부]라는 변수가 만족도와 참여도에 미친 영향을 종합적으로 분석.

## 2. 입력 데이터 구조 (Input Batch)
- 행별 데이터셋: {참석여부, 성함, 성별, 국적, 학번, 느낀 점}

## 3. 프롬프트 엔진 지시 사항 (System Instruction)
- **Role**: 설문 통계 기반 인사이트 리포트 작성가
- **Analysis Task**:
  1. **참석 여부별**: 참석자와 미참석자의 만족도 차이 및 원인 분석
  2. **국적별**: 언어 능력이나 소통 방식에 따른 활동 적응도 차이 도출
  3. **성별별**: 활동 성향(적극성) 및 참여 방식에서의 성별 간 특징 비교

## 4. 예상 출력 예시 (Expected Output)
"이번 행사에서 참석자들은 대체로 유익했다는 평가를 내린 반면, 미참석자들은 활동의 가치를 크게 느끼지 못하는 경향을 보였습니다. 국적별로는 언어 장벽이 낮았던 일본 학생들이 적극적으로 참여한 반면, 중국 학생들은 한국어 소통에 어려움을 느껴 활동 이해도가 낮았습니다. 성별에 따라서는 여학생들의 참여도가 매우 높았으나, 남학생들은 참여 방식이 복잡해 적극성이 다소 떨어지는 패턴이 관찰되었습니다."
